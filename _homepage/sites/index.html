In 30 Sekunden starten

Clone, installieren, läuft. Kein DB-Setup.

Quickstart 
git clone https://github.com/McCryptoX/flatwiki && cd flatwiki && ./install.sh

Check 
docker compose ps

Danach öffnen: http://127.0.0.1:3000/setup (erster Start)
